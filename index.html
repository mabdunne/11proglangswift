When? Where? Who?

  • When - Open source Dec 2015. Stable release March 2017
  • Where - Apple
  • Who - Chris Lattner, founder and project lead, started coding on his own July 2010.

What Purpose Does Language Serve?

  • Introduced as "Objective-C without the C".
  • For mobile application development, on iOS & linux.
  • Cleaner (reduce useless syntax), safer (null/nil handling).
  • Swift Playgrounds make programming interactive, approachable and fun.
  • Swift Playgrounds were first introduced as a way to experiment with your code in Xcode.

		var outdoorTemperatureInFahrenheit = 88.2

Details

  • Compiled Language - compiled to machine code that runs on the Objective-C runtime.
  • Strongly Typed - checks variable/argument types at compile time.
  • Imperative - object oriented imperative language with some tools for programming functionally.
